## Install Angular CLI

```bash
pnpm i -g @angular/cli
```

## Create Angular App

```bash
ng new <angular-app-name>
```

## Launch Dev Server

In`package.json` configure IP addy and assign avail port:

```json
"start": "ng serve --host 192.168.5.555 --port 5555",
```

Launch development server:

```bash
pnpm start
```
